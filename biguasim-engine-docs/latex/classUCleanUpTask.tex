\doxysubsection{Detailed Description}
\mbox{\hyperlink{classUCleanUpTask}{UClean\+Up\+Task}} Inherits from the Task\+Sensor class. Initializes the clean up task in the world. This only works in the Clean\+Up world. The reward is based on the number of pieces of trash placed in the trash can. For each piece of trash added to the can, a reward of 1 is given. For each piece of trash removed, a reward of -\/1 is given. If all the trash is in the can, terminal is given.
