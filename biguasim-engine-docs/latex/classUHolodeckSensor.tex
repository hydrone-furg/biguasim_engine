Holodeck\+Sensor Abstract base class for sensors within holodeck Handles publishing sensor data To function properly, Holodeck\+Sensors must be attached to an agent that is controlled by a holodeck controller. To override, you must implement\+: Get\+Data\+Key Get\+Data\+Length Tick\+Sensor\+Component And in Begin\+Play you must call Super\+::\+Begin\+Play But you cannot override\+: Tick\+Component
