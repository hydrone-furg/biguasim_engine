\doxysubsection{Detailed Description}
Add\+Sensor\+Command Command used to add a sensor to an agent Warning\+: This command is meant for initialization. Adding a sensor with the same name as a previously removed sensor may cause problems.

String\+Parameters expects five arguments\+: the agent name, sensor name, sensor class, sensor parameters, and socket. Number\+Parameters expects six arguments\+: locations x, y, and z and rotations pitch, yaw, and roll.
