\doxysubsection{Detailed Description}
Custom\+Command This command is inherited by a blueprint. It is to be used to easily implement simple commands for any number of different functions.

String\+Parameters expect at least 1 argument as the name of the command and any number of additional string params. Number\+Parameters of any size.
