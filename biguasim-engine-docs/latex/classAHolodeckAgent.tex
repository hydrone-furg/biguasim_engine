\doxysubsection{Detailed Description}
\mbox{\hyperlink{classAHolodeckAgent}{AHolodeck\+Agent}} The base class for holodeck agents. Holodeck\+Agents are controllable from python. A Holodeck\+Agent contains the logic for how it acts at a low level. To get to a higher level of control, the logic should be implemented in a Holodeck\+Control\+Scheme.
