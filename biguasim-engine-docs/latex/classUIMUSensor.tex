\doxysubsection{Detailed Description}
Simulated Inertial Measurement Unit (IMU). Inherits from the Holodeck\+Sensor class. Calculates and returns a 6-\/DOF data array representing local linear acceleration and angular velocity (gyroscope). Simulates real-\/world sensor imperfections including Gaussian noise and Random Walk Bias accumulation.
