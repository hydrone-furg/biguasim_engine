\doxysubsection{Detailed Description}
\mbox{\hyperlink{classUCommandCenter}{UCommand\+Center}} It subscribes a memory space to receive commands from the client binding. It expects the memory to be formatted in UTF-\/8 JSON. When creating a new commandcenter object, you must call the init after or it will not function. It requires a valid pointer to the server to be initialized, and the currrent gamemode object to continue running correctly.
