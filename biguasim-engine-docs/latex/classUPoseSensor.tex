Main tick function calculating the Transformation Matrix. 

Main execution loop that calculates the 4x4 Homogeneous Transformation Matrix. Retrieves the Forward, Right, Up vectors, and the Location of the sensor. Negates the Right vector to convert from Unreal\textquotesingle{}s Right-\/\+Handed to the client\textquotesingle{}s Left-\/\+Handed coordinate system, and packs everything into a 16-\/element array.
