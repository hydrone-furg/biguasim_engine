\doxysubsection{Detailed Description}
\mbox{\hyperlink{classULocationTask}{ULocation\+Task}} Inherits from the Task\+Sensor class. Calculates a sparse distance reward. Maximizes Distance if Maximize distance is set to true. Location is used when actor is null. Terminal is set to true when the agent is within its Goal\+Distance.
