Main tick function calculating the Forward, Left, and Up vectors. 

Main execution loop that calculates the orientation matrix. Retrieves the Forward, Right, and Up vectors of the sensor, negates the Right vector to get the Left vector, and populates a 9-\/element array (3x3 matrix) representing the absolute orientation in the client\textquotesingle{}s coordinate system.
