\doxysubsection{Detailed Description}
Unmanned Aerial Vehicle (UAV) agent representing a DJI Matrice drone. 


\begin{DoxyItemize}
\item Inherits from the Holodeck\+Agent class (Note\+: not buoyant). On any tick this object will apply the given forces. Desired values must be set by a controller. 
\end{DoxyItemize}
