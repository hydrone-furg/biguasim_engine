\doxysubsection{Detailed Description}
Core communication server for Bigua\+Sim. This class resides in the engine and handles the passing of messages to the Python client through shared memory. A new shared memory block is created for each sensor, action space, and setting. 


\begin{DoxyItemize}
\item There should only be one \mbox{\hyperlink{classUHolodeckServer}{UHolodeck\+Server}} active at a time, instantiated by the Holodeck\+Game\+Instance. 
\end{DoxyItemize}
