\doxysubsection{Detailed Description}
\mbox{\hyperlink{classUAvoidTask}{UAvoid\+Task}} Inherits from the Task\+Sensor class. Calculates Avoid reward based on distance and line of sight. If Only\+Within\+Sight is true, the reward is set to the percent distance covered from the Min\+Distance to the To\+Avoid Target if the angle from the agent to the Target is less than FOVRadians and is there is nothing blocking the agent\textquotesingle{}s line of sight, otherwise the reward is 0. If Onlywithin\+Sight is false, the reward is set to the percent distance covered from the Min\+Distance to the To\+Avoid Actor. Terminal is always false.
