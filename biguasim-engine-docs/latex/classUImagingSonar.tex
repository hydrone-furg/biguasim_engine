\doxysubsection{Detailed Description}
Forward Looking Imaging Sonar (FLS) sensor simulation. Emits an acoustic wave, calculates intersections using an \mbox{\hyperlink{classOctree}{Octree}}, and simulates complex acoustic phenomena including multipath reflections, speckle noise, and acoustic shadowing. Supports generating ground-\/truth point clouds and elevation maps.
