Main tick function calculating position and handling depth signal loss. 

Main execution loop that calculates the global position. Simulates GPS signal attenuation in water\+: if the sensor\textquotesingle{}s depth exceeds GPSDepth, it returns NaN (Not a Number) for all coordinates. Otherwise, it applies multivariate normal noise and writes the XYZ coordinates to the shared memory buffer.
