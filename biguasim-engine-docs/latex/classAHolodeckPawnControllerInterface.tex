\doxysubsection{Detailed Description}
\mbox{\hyperlink{classAHolodeckPawnControllerInterface}{AHolodeck\+Pawn\+Controller\+Interface}} A controller for \mbox{\hyperlink{classHolodeck}{Holodeck}} Agents. If a Holodeck\+Agent doesn\textquotesingle{}t contain this controller or a controller which inherits it, then you will run into problems. This class handles the subscribing of sensors and setting up action buffer channels. The sensors themselves tell the controller to subscribe them. Its base classes must report what size of action buffer they need. Holodeck\+Pawn\+Controllers get the data from the shared memory and give the commands to the pawns/agents.
