Ticks the game mode. Checks for changes in settings, handles reset signals, and ticks the game instance and command center. 

Called every frame to update the simulation state. Propagates the tick to the Game\+Instance and Command\+Center. It also listens for the shared memory Reset\+Signal; if triggered by the Python client, it reloads the current level.
