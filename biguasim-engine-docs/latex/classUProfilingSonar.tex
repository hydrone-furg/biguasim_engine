Parses JSON parameters to set dynamic resolutions, enforcing profiling acoustic limits. 

Parses the JSON configuration sent by the Python client. Intercepts the configuration to enforce the physical characteristics of a Profiling Sonar (extremely narrow 1-\/degree elevation beam, and specific min/max ranges) before passing the payload to the parent Imaging\+Sonar class. Also sets high-\/resolution default bins if the user doesn\textquotesingle{}t provide them.
