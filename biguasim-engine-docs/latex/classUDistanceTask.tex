\doxysubsection{Detailed Description}
\mbox{\hyperlink{classUDistanceTask}{UDistance\+Task}} Inherits from the Task\+Sensor class. Calculates a dense distance based reward. Maximizes Distance if Maximize distance is set to true. Location is used when actor is null. Terminal is set to true when the agent is within its Goal\+Distance.
