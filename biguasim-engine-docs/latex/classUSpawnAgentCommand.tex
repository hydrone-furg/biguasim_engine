\doxysubsection{Detailed Description}
Spawn\+Agent\+Command The command used to spawn agents in the world.

String\+Parameters are expected to be Agent\+Type, and then Name. Number\+Parameters are expected to be X, Y, then Z coords of the location to spawn at.

To enable it to spawn other agents you will need to\+:
\begin{DoxyEnumerate}
\item Go to the Holodeck\+Game\+Mode\+Blueprint in editor
\item Go to the Spawn\+Agent Function and select the Agent\+Bp\+Map data field
\item Add the agent\textquotesingle{}s name and bp reference to the map 
\end{DoxyEnumerate}
