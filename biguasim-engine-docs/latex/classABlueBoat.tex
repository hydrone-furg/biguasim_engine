\doxysubsection{Detailed Description}
Surface vessel agent (Blue\+Boat) for multi-\/domain simulation. 


\begin{DoxyItemize}
\item Inherits from the Holodeck\+Buoyant\+Agent class. On any tick this object will apply the given forces. Desired values must be set by a controller. 
\end{DoxyItemize}
