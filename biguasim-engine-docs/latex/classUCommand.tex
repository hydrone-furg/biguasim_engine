\doxysubsection{Detailed Description}
\mbox{\hyperlink{classUCommand}{UCommand}} This is the abstract base class for commands in the holodeck When instantiating an inherited command, you must call the Init function.
\begin{DoxyItemize}
\item Note\+: commands are more safely instantiated from the commandfactory. When inheriting from this class, you must\+:
\item Implement the execute function.
\item Add the inhereted class and its name string to the \mbox{\hyperlink{CommandFactory_8h}{Command\+Factory.\+h}} file. 
\end{DoxyItemize}
