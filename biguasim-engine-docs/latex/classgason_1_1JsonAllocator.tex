\doxysubsection{Detailed Description}
Automatic memory manager for parsed \mbox{\hyperlink{structgason_1_1JsonValue}{Json\+Value}}. \begin{DoxySeeAlso}{See also}
\mbox{\hyperlink{classgason_1_1JsonAllocator_a92a06da4ffcbc7014bd63b175e531685}{json\+Parse()}}. used internally by \mbox{\hyperlink{classgason_1_1JsonAllocator_a92a06da4ffcbc7014bd63b175e531685}{json\+Parse()}}. you do not need to manually handle this object. just keep allocator instance as long as you want to keep Json\+Values.
