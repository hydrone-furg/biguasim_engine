Range Finder Sensor (1D to 2D Lidar). Inherits from the Holodeck\+Sensor class. Gets distances to the first collision. By default, returns a single float representing the distance in the agent\textquotesingle{}s forward direction. Increasing Laser\+Count creates an array of rays evenly distributed 360 degrees in a plane around the agent. Laser\+Angle offsets each of these rays, transforming the plane into a conical shape.
